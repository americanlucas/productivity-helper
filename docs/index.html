Recursos Principais

💾

Salvamento de Links

Salve páginas interessantes com um clique. Acesse rapidamente seus links favoritos organizados por data.

📝

Anotações Rápidas

Capture ideias instantaneamente. Salve texto selecionado de qualquer página como nota pessoal.

✅

Gerenciamento de Tarefas

Organize suas tarefas diárias. Adicione, complete e gerencie suas atividades sem sair do navegador.

⚡

Atalhos de Teclado

Trabalhe mais rápido com atalhos intuitivos. Ctrl+S para salvar páginas, Ctrl+Shift+N para notas.

🎯

Menu de Contexto

Clique com o botão direito para salvar links ou texto selecionado diretamente no menu de contexto.

📊

Estatísticas de Uso

Acompanhe sua produtividade com estatísticas detalhadas de links, notas e tarefas salvos.

Como Funciona

1

Instale a Extensão

Baixe e instale a extensão no Chrome seguindo as instruções simples.

2

Clique no Ícone

Acesse o popup clicando no ícone da extensão na barra de ferramentas.

3

Organize-se

Use as abas para gerenciar links, notas e tarefas de forma organizada.

Instalação

📥 1. Baixar

Baixe o arquivo ZIP da extensão usando o botão de download.

📂 2. Extrair

Extraia o conteúdo do arquivo ZIP em uma pasta no seu computador.

🔧 3. Instalar

No Chrome, vá em Extensões → Modo do desenvolvedor → Carregar sem compactação.

✅ 4. Usar

A extensão aparecerá na barra de ferramentas. Clique para começar a usar!
